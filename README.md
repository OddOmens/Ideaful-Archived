# Ideaful

**Note: This project is archived and no longer supported.**

Ideaful is an iOS application designed to help you manage your ideas, tasks, notes, and creative projects.

## Getting Started

### Prerequisites

- Xcode (latest version recommended)
- iOS 17.0+ (Requires modern SwiftUI support)

### Installation

1. Clone the repository.
2. Open `Ideaful.xcodeproj` in Xcode.
3. Wait for Swift Package Manager to resolve dependencies (if any).
4. Build and run on your simulator or device.

## Features

- **Idea Management**: Organize your creative ideas with statuses and tags.
- **Task Tracking**: Manage tasks associated with your ideas.
- **Notes**: Keep detailed notes for your projects.
- **Statistics & Achievements**: Track your progress (some features may be disabled in the archive version).
- **Dashboard**: Overview of your active ideas and tasks.
