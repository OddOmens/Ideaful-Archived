## Getting Started

### First Launch
When you first open Ideaful, you'll see the Dashboard view with your ideas organized by status. The app uses iCloud sync to keep your data synchronized across all your Apple devices.

### Navigation
- **Dashboard**: Main view showing all your ideas grouped by status
- **Settings**: Access customization, themes, and app preferences
- **All Tasks**: View all uncompleted tasks across all ideas

---

## Core Features

### Dashboard
The Dashboard is your central hub for managing ideas:
- Ideas are automatically grouped by status (Unassigned, Planning, Started, On Hold, Completed, Cancelled, Archived)
- Tap any idea to view its details
- Use the menu button (☰) to access management options
- Tag filter button allows filtering ideas by tags
- Quick access to all tasks via the list icon

### Quick Actions
- **Swipe Left** on an idea: Delete
- **Swipe Right** on an idea: Quick status changes (Complete, Start, Hold)
- **Tap** an idea: View full details
- **Long press**: Additional context menu options

---

## Ideas Management

### Creating an Idea
1. Tap the **+** button in the top right
2. Enter a **Title** (required, max 85 characters)
3. Add a **Description** (optional, max 85 characters)
4. Select a **Status** (defaults to "Unassigned")
5. Add **Tags** for organization
6. The idea saves automatically as you type

### Idea Details
Each idea contains:
- **Title**: The name of your idea (85 character limit)
- **Description**: Short summary of the idea (85 character limit)
- **Status**: Current workflow stage
- **Tags**: Color-coded labels for organization
- **Images**: Up to 10 inspirational images
- **Notes**: Unlimited detailed notes
- **Tasks**: Action items to complete the idea

### Editing Ideas
- Tap any field to edit directly
- Changes save automatically
- Character counters show remaining space
- Press Return/Enter to save and dismiss keyboard

### Adding Images
1. In idea details, scroll to "Inspirational Images"
2. Tap the **+** button
3. Select up to 10 images from your photo library
4. Tap an image to view full screen
5. Long press or use context menu to delete images

### Deleting Ideas
- Swipe left on an idea in the Dashboard
- Or tap the trash icon in idea details
- Confirmation dialog appears (can be disabled in settings)

## Tasks System

### Task Overview
Tasks help you break down ideas into actionable steps. Each task can have:
- Title and description
- Due date
- Reminder notification
- Priority level (None, Low, Medium, High)
- Completion status

### Creating Tasks

#### Quick Task Creation
1. Open an idea
2. Tap the list icon in the top right
3. Type in the "New task..." field at the bottom
4. Tap **+** to create instantly

#### Detailed Task Creation
1. Open an idea
2. Tap the list icon
3. Tap the **+** button (when field is empty)
4. Fill in:
   - **Title**: Task name
   - **Description**: Additional details
   - **Priority**: None/Low/Medium/High
   - **Due Date**: Optional deadline
   - **Reminder**: Optional notification time

### Task Priorities
- **High Priority** (Red flag): Urgent tasks
- **Medium Priority** (Orange flag): Important tasks
- **Low Priority** (Yellow flag): Nice-to-have tasks
- **No Priority** (Gray): Unassigned priority

### Task Filtering
The task view includes smart filters:
- **Today**: Tasks due today (shows overdue count)
- **Tomorrow**: Tasks due tomorrow (shows this week count)
- **High/Med/Low**: Filter by priority level
- Tap a filter to activate, tap again to clear

### Task Overview Cards
Two collapsible overview sections:
1. **Task Overview**: Shows Today and Tomorrow counts
2. **Task Priorities**: Shows High, Medium, and Low priority counts

### Managing Tasks
- **Toggle completion**: Tap the checkbox
- **Edit task**: Tap the task row
- **Delete task**: Swipe left
- **Clear completed**: Tap the trash icon in toolbar
- **Collapse/expand overview**: Tap chevron in toolbar

### All Tasks View
Access from Dashboard to see all uncompleted tasks:
- Tasks grouped by idea
- Same filtering options as individual task views
- Tap idea name to jump to that idea
- Complete or edit tasks directly

### Task Notifications
When you set a reminder:
- Device notification appears at the specified time
- Notification includes idea name and task title
- Requires notification permissions (prompted on first use)

## Tags & Organization

### Creating Tags
1. Tap the tag icon in Dashboard toolbar
2. Tap **+** in the tag filter bar
3. Or go to Settings → Tags
4. Enter tag name
5. Choose a color from 12 predefined options
6. Tap Save

### Available Tag Colors
- Blue (#4B7BEC)
- Green (#26DE81, #20BF6B)
- Red (#FC5C65, #FF6B6B)
- Orange (#FD9644)
- Purple (#A55EEA, #5F27CD)
- Teal (#2BCBBA, #00D2D3)
- Pink (#E456F0)
- Light Blue (#54A0FF)

### Using Tags
- **Add to ideas**: Tap tags section in idea details
- **Filter by tags**: Use tag filter bar in Dashboard
- **Multiple tags**: Ideas can have unlimited tags
- **Tag display**: Shows as colored pills with tag name

### Tag Filtering
1. Tap tag icon in Dashboard
2. Tag filter bar appears at top
3. Tap tags to filter (multiple selection supported)
4. Tap **X** to clear all filters
5. Only ideas with selected tags appear

### Bulk Tagging
1. Tap menu (☰) → Bulk Tag Ideas
2. Select multiple ideas by tapping
3. Tap tag icon in toolbar
4. Choose tags to apply
5. Tags added to all selected ideas

### Managing Tags
Settings → Tags:
- View all tags
- Edit tag names and colors
- Delete unused tags
- See tag usage count

Ideaful includes 7 built-in statuses:

5. **Completed** (Green): Finished ideas

- Swipe right on an idea

- Open the idea

- Select new status from list

**Method 3: Bulk Archive**

### Status Visibility
Control which statuses appear in Dashboard:

- Toggle "Show Archived Ideas"

### Custom Statuses
Settings → Statuses:
- Enable/disable default statuses
- Customize status names
- Change status colors
- Reorder status display

### Status Sections
- Dashboard groups ideas by status
- Tap disclosure arrow to expand/collapse
- Count badge shows number of ideas per status
- Empty statuses are hidden automatically

## Data Management

### iCloud Sync
- Automatic synchronization across devices
- Requires iCloud account
- Data syncs in real-time
- Handles conflicts automatically

### Export Options
Settings → Export Ideas:

#### Export Formats
1. **Text File (.txt)**
   - Plain text format
   - Grouped by status
   - Includes title, description, notes

2. **CSV File (.csv)**
   - Spreadsheet compatible
   - Columns: Status, Title, Description, Notes, Update Date
   - Import into Excel, Numbers, Google Sheets

3. **Markdown File (.md)**
   - Formatted text with headers
   - Compatible with note apps
   - Preserves structure and hierarchy

#### Export Process
1. Settings → Export Ideas
2. Choose format
3. Tap export button
4. Share via system share sheet
5. Save to Files, email, or other apps

### Mass Delete
Settings → Mass Delete Ideas:
- Deletes ALL ideas permanently
- Confirmation required
- Cannot be undone
- Use with extreme caution

### Data Storage
- All data stored locally in CoreData
- Images saved in app documents directory
- Automatic migration between app versions
- Efficient storage management

## Privacy & Data

### Data Privacy
Ideaful is designed with your privacy as a top priority:
- **No Data Collection**: The app does not collect, track, or transmit your personal data to any servers
- **No Analytics**: No usage analytics or tracking services are embedded in the app
- **No Third-Party Services**: Your data is never shared with third parties
- **Local Storage**: All your ideas, tasks, and notes are stored locally on your device
- **Your Data, Your Control**: You have complete ownership and control of your information

### iCloud Sync
When iCloud sync is enabled:
- **Apple's Infrastructure**: Data syncs through Apple's iCloud service
- **End-to-End Encryption**: iCloud data is encrypted in transit and at rest
- **Apple's Privacy Policy**: Subject to Apple's iCloud terms and privacy policy
- **Optional Feature**: You can disable iCloud sync in device settings
- **Device-Only Option**: App works fully without iCloud if preferred

### Data Security
- **Local Encryption**: Data stored in iOS secure container
- **No Account Required**: No login, registration, or account creation needed
- **Offline Capable**: Full functionality without internet connection
- **Secure Storage**: Images and data protected by iOS security features
- **No Passwords Stored**: App doesn't handle or store any passwords

### What Data is Stored
The app stores only what you create:
- **Ideas**: Titles, descriptions, statuses, creation/update dates
- **Tasks**: Task details, due dates, priorities, completion status
- **Notes**: Note titles and content
- **Tags**: Tag names and colors
- **Images**: Photos you add to ideas (stored locally)
- **Settings**: Your app preferences and customization choices
- **Achievements**: Progress tracking for in-app achievements

### What Data is NOT Collected
Ideaful does NOT collect or access:
- ❌ Personal identifying information
- ❌ Location data
- ❌ Contact information
- ❌ Usage analytics or behavior tracking
- ❌ Device identifiers for advertising
- ❌ Browsing history or app usage patterns
- ❌ Any data outside the app's sandbox

### Notifications & Permissions
- **Notification Permission**: Only used for task reminders you set
- **Photo Library Access**: Only when you choose to add images to ideas
- **No Background Tracking**: App doesn't track activity when closed
- **Minimal Permissions**: Only requests permissions for features you use

### Data Portability
You maintain full control of your data:
- **Export Anytime**: Export all ideas in multiple formats (Text, CSV, Markdown)
- **No Lock-In**: Your data is exportable and readable outside the app
- **Standard Formats**: Export uses universal file formats
- **Easy Migration**: Take your data with you if you switch apps

### Data Deletion
Complete control over data removal:
- **Individual Deletion**: Delete specific ideas, tasks, or notes anytime
- **Mass Delete**: Remove all ideas at once via Settings
- **Permanent Removal**: Deleted data is permanently removed from device
- **iCloud Cleanup**: Deleting from device also removes from iCloud sync
- **No Server Copies**: No data retained on external servers

### Compliance
- **GDPR Friendly**: No personal data collection means no GDPR concerns

## Frequently Asked Questions

### General Questions

**Q: Does Ideaful require an internet connection?**
A: No, Ideaful works fully offline. Internet is only needed for iCloud sync across devices.

**Q: Can I use Ideaful on multiple devices?**
A: Yes! With iCloud sync enabled, your data automatically syncs across all your Apple devices (iPhone, iPad, Mac if available).

**Q: Is there a subscription fee?**
A: No, Ideaful is a one-time purchase with no recurring fees or subscriptions.

**Q: How many ideas can I create?**
A: Unlimited! There's no cap on ideas, tasks, notes, or tags.

**Q: Can I import data from other apps?**
A: Currently, Ideaful doesn't support direct import, but you can manually create ideas from exported data from other apps.

### Privacy & Security

**Q: Is my data safe?**
A: Yes, all data is stored locally on your device using iOS secure storage. If using iCloud, data is encrypted in transit and at rest.

**Q: Can Odd Omens see my ideas?**
A: No, we have no access to your data. Everything stays on your device or in your personal iCloud account.

**Q: What happens if I delete the app?**
A: Local data is deleted. If you had iCloud sync enabled, data remains in iCloud for 30 days before permanent deletion.

**Q: Can I use Ideaful without iCloud?**
A: Yes, the app works perfectly without iCloud. You'll just lose cross-device sync capability.

### Features & Usage

**Q: What's the difference between Notes and Description?**
A: Description is a short summary (85 chars) shown on the Dashboard. Notes are unlimited detailed documentation within each idea.

**Q: Can I share ideas with others?**
A: You can export ideas and share the exported file. There's no built-in collaboration feature.

**Q: How do I backup my data?**
A: Use the Export feature (Settings → Export Ideas) to create backups in Text, CSV, or Markdown format.

**Q: Can I customize the statuses?**
A: Yes! Go to Settings → Statuses to enable/disable, rename, and recolor statuses.

**Q: What's the maximum number of images per idea?**
A: You can add up to 10 images per idea.

### Technical Questions

**Q: What iOS version is required?**
A: Check the App Store listing for current requirements. The app is optimized for recent iOS versions.

**Q: Does Ideaful work on iPad?**
A: Yes, Ideaful is a universal app that works on both iPhone and iPad.

**Q: How much storage does the app use?**
A: The app itself is small. Storage usage depends on how many ideas and images you create.

**Q: Can I use Ideaful on Mac?**
A: Check the App Store for Mac availability. If available, it syncs seamlessly with iOS versions via iCloud.

### Troubleshooting

**Q: My ideas aren't syncing between devices. What should I do?**
A: Ensure iCloud Drive is enabled in Settings → [Your Name] → iCloud → iCloud Drive. Check that you're signed into the same iCloud account on all devices.

**Q: I accidentally deleted an idea. Can I recover it?**
A: Unfortunately, deleted ideas cannot be recovered. We recommend regular exports as backups.

**Q: The app is running slowly. How can I improve performance?**
A: Try closing and reopening the app, restarting your device, or deleting old/unused ideas and images.

**Q: Notifications aren't working. What's wrong?**
A: Check Settings → Notifications → Ideaful and ensure notifications are enabled. Also verify reminder times are set in the future.
